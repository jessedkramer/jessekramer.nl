Deze website is mijn eigen plek op het internet. Op de [Over mij](/about)-pagina vertel ik wie ik ben en waar ik mee bezig ben. Daarnaast vind je hier mijn [Journal](/journal), waarin ik ontwikkelingen, ervaringen en andere onderwerpen deel die mij bezighouden.

## Van idee naar website

Ik wilde al een tijd een nieuwe persoonlijke website bouwen. Tegelijkertijd wilde ik meer leren over AI. Uiteindelijk heb ik die twee ideeën samengebracht.

Het begon met een idee dat ik op X deelde over een **AI-native CMS**.

{{tweet:2073911293376082334}}

Dat idee heb ik vervolgens in de praktijk gebracht. De website die je nu ziet, heb ik gebouwd met behulp van **ChatGPT** en **Cursor**. In plaats van een traditioneel CMS wordt vrijwel alle content beheerd vanuit een GitHub-repository. AI begrijpt die structuur en helpt daardoor bij het schrijven en beheren van pagina's, waaronder deze **Journal**.

Ook de **Currently**-balk bovenaan de website wordt op deze manier beheerd. Die laat in één oogopslag zien waar ik op dat moment mee bezig ben of wat er speelt.

Onderweg heb ik veel geleerd over GitHub en Vercel, maar ook over AI. Waar ik ChatGPT eerst vooral gebruikte als chatbot, gebruik ik AI inmiddels als hulpmiddel om ideeën uit te werken, code te schrijven en projecten te bouwen.

## From idea to website

I had wanted to build a new personal website for some time. At the same time, I wanted to learn more about AI, so I decided to combine the two.

It began with an idea I shared on X about an **AI-native CMS**.

{{tweet:2073911293376082334}}

I then put that idea into practice. The website you are viewing was built with the help of **ChatGPT** and **Cursor**. Instead of using a traditional CMS, nearly all content is managed through a GitHub repository. AI understands that structure and can therefore help write and manage pages, including this **Journal**.

The **Currently** bar at the top of the website is managed in the same way. It gives a quick overview of what I am doing or what is currently relevant.

Along the way, I learned a great deal about GitHub and Vercel, as well as AI. I used to see ChatGPT mainly as a chatbot, but I now use AI as a tool for developing ideas, writing code and building projects.
